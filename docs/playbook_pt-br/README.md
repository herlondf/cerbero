# Playbook do Cerbero

Documentação técnica da biblioteca de autenticação JWT Cerbero.

---

## Conteúdo

| Seção | Status |
|-------|--------|
| [01-visao-geral](./01-visao-geral/) — Arquitetura, objetivos de design e como o Cerbero se encaixa na Família Olímpica | Disponível |
| [02-jwt](./02-jwt/) — Internos do JWT: header, payload, assinatura, HS256 com a RTL do Delphi | Disponível |
| [03-api-keys](./03-api-keys/) — Geração, rotação e validação de API Keys | Planejado |
| [04-oauth2](./04-oauth2/) — Fluxos OAuth2 com integração ao Iris | Planejado |

---

## Ordem de Leitura

Se você é novo no Cerbero, leia as seções em ordem:

1. **01-visao-geral** — entenda o escopo e os não-objetivos da biblioteca.
2. **02-jwt** — aprenda como os tokens são construídos, assinados e verificados.
3. Demais seções conforme as funcionalidades forem lançadas.

---

## Contribuindo com o Playbook

A documentação vive junto com o código. Ao adicionar ou alterar uma funcionalidade, atualize a seção correspondente do playbook no mesmo pull request. Consulte [../CONTRIBUTING_pt-br.md](../CONTRIBUTING_pt-br.md) para o guia completo de contribuição.
